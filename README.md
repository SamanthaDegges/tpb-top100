# tpb-top100

*List the top 100 most popular movies on The Pirate Bay.*


## Purpose

This is a very simplistic NPM package designed as a teaching aide for a talk I'm
giving @ [CodingHouse][].


## Installation

You can install this package directly from NPM:

```console
$ npm install -g tpb-top100
```
